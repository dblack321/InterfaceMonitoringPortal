# InterfaceMonitoringPortal

--Purpose--
The Interface Monitoring Portal is an application for Innovia Films Ltd which monitors the status of all the scheduled tasks to tranfer data between the business systems by flashing alerts onto the application for users to review. These alerts will be created when user-defined thresholds have had the limits breached.

NOTE: Due to Innovia Films Ltd interface data being sensitive, all code and functionality connecting this application to Innovia Films data has been removed.

--System Requirements--
Python installed
All python packages listed in requirements.txt installed
Internet connection
Web browser

--Installation / Startup--
Install packages using: pip install -r requirements.txt
Run the development environment using: py manage.py runserver

--How to use--
Login to the application using one of 2 credentials
  - Username: user, Password: password (User privaleges)
  - Username: admin, Password: password (Admin privaleges)
Landing page will be the home page
Use the navigation bar at the top of application to naviagte between
  - Home
  - Masterdata
  - About
